# Lens 3 — Macro Context

## Core question

Does the current macro regime help or hurt THIS specific thesis right now —
not macro commentary in general.

## What to produce in this section

1. **Rate path.** Current policy stance and market-implied path; how this
   name's duration profile reacts (long-duration growth vs. cash-generative
   value; refinancing needs).
2. **Capex cycle.** For AI/semi-adjacent names: where hyperscaler capex
   guidance currently stands (the master leading indicator for the chain).
   For other sectors: the equivalent end-demand cycle.
3. **Policy / tariff / export controls.** Any live tariff regimes, export
   bans (especially China exposure for semis), subsidies, or executive
   actions that touch this company's inputs, customers, or geography.
4. **Geopolitics & shocks.** War, supply shocks, elections — only if they
   have a concrete transmission path to this name. Note the frontrunning
   trap: a shock that already spiked the sector to highs is late; shocks
   create entries in quality names that sold off indiscriminately.
5. **FX/commodities** only when material (e.g. input feedstock pricing,
   revenue currency mismatch).

## Output

3–5 bullets, each naming the macro factor, its direction, and its concrete
transmission to this stock. End with one line: net macro tailwind, headwind,
or neutral for this thesis over the next 6–12 months.
